# E-Commerce AI Agent

Complete price comparison tool using Flask + GPT-4o + Bright Data.

## Quick Start
1. Install: pip install -r requirements.txt
2. Configure: Edit .env file
3. Run: python app.py
4. Open: http://localhost:8000

## Features
- Real-time price comparison
- 7+ e-commerce platforms
- AI-powered search
- Beautiful UI
